Exercices pratiques RGPD

Ces exercices vous permettent de mettre en pratique toutes les notions vues : collecte, consentement, sécurisation et droits utilisateur.

Exercice 1 : Créer un formulaire conforme RGPD

Créer un formulaire demandant nom, email et consentement :

<form id="exerciseForm">
  <label for="name">Nom:</label>
  <input type="text" id="name" required>

  <label for="email">Email:</label>
  <input type="email" id="email" required>

  <input type="checkbox" id="consent" required>
  <label for="consent">J'accepte le RGPD</label>

  <button type="submit">Envoyer</button>
</form>

Correction JS :

const form = document.getElementById('exerciseForm');
form.addEventListener('submit', e => {
  e.preventDefault();
  const name = document.getElementById('name').value;
  const email = document.getElementById('email').value;
  const consent = document.getElementById('consent').checked;

  if(!consent) {
    alert("Consentement obligatoire !");
    return;
  }

  localStorage.setItem('exerciseUser', JSON.stringify({ name, email }));
  alert("Formulaire soumis avec succès !");
});

Exercice 2 : Créer un popup de consentement

Afficher un modal RGPD au chargement de la page :

<div id="modal">
  <p>Acceptez-vous le traitement de vos données ?</p>
  <button id="accept">Accepter</button>
  <button id="decline">Refuser</button>
</div>

Correction JS :

const modal = document.getElementById('modal');
const accept = document.getElementById('accept');
const decline = document.getElementById('decline');

if(!localStorage.getItem('rgpdConsent')) {
  modal.style.display = 'block';
}

accept.addEventListener('click', () => {
  localStorage.setItem('rgpdConsent', true);
  modal.style.display = 'none';
});

decline.addEventListener('click', () => {
  localStorage.setItem('rgpdConsent', false);
  modal.style.display = 'none';
});

Exercice 3 : Droits utilisateur

Afficher, modifier et supprimer les données stockées :

const userData = JSON.parse(localStorage.getItem('exerciseUser'));
console.log(userData);

// Modifier
userData.email = "nouvel@email.com";
localStorage.setItem('exerciseUser', JSON.stringify(userData));

// Supprimer
localStorage.removeItem('exerciseUser');

Exercice 4 : Sécurisation minimale

  • Valider tous les champs côté client.
  • Stocker les données sensibles uniquement côté serveur (ici pédagogique avec localStorage).
  • Utiliser HTTPS pour la page réelle.
